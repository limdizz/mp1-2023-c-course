# Указатели

![](../../images/pointers.png)

В этой простенькой задаче вам нужно реализовать следующие функции

1. `Swap` - содержимое по адресу `x` записывает по адресу `y`, и наоборот :smile:
2. `PreIncrement` - реализует `++i`
3. `PostIncrement` - реализует `i++`
4. `Exchange` - записывает `new_value` по адресу `x`, возвращает старое значение до обновления.
